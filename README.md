# uptask-front

## Tecnologias
+ NodeJS
+ TypeScript
+ Vite
+ Tailwindcss

## Bibliotecas
+ React Router Dom
+ React Toastify
+ React Hook Form
+ Zod

### Instrucciones
1.- Cree el archivo `.env.local` para las variables de entorno de manera local.

2.- Agregar la variable de entorno para la conexion con el backend
```
VITE_API_URL
```
